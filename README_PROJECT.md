# 📱 SOS Automotriz - Interfaz Premium de Asistencia (Frontend)

Frontend de última generación para la plataforma SOS Automotriz. Una interfaz minimalista, profesional y altamente responsiva diseñada para conectar a conductores en emergencia con talleres especializados.

## ✨ Experiencia de Usuario
- **🗺️ Rastreo en Vivo**: Seguimiento en tiempo real de la ubicación del técnico mediante Leaflet.js.
- **🛡️ Panel de Despacho IA**: Gestión centralizada de solicitudes con visualización de diagnósticos detallados generados por IA.
- **📊 Matriz de Poder**: Control granular de roles y permisos con una interfaz de administración avanzada.
- **💸 Pasarela de Pagos**: Flujo optimizado para pagos vía QR, Tarjeta y Efectivo con liquidación instantánea.
- **🌗 Diseño Premium**: Estética moderna con efectos de cristal (glassmorphism), tipografía técnica y micro-animaciones.

## 🛠️ Tecnologías
- **Framework**: Angular 17+ (Standalone Components)
- **Estilos**: Tailwind CSS & Vanilla CSS (Custom Design System)
- **Mapas**: Leaflet API
- **Iconos**: Bootstrap Icons
- **Estado**: RXJS & Services

## 🔧 Instalación
1. Clonar el repositorio.
2. Instalar módulos: `npm install`.
3. Iniciar entorno de desarrollo: `ng serve`.
4. Acceder a `http://localhost:4200`.
